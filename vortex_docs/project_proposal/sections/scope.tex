\section{Scope of the Project}
Shows what is possible and what is not.

\subsection{In Scope}
What we can do:

\begin{itemize}
  \item Steam + Epic (initially)
  \item Read-only, easily accessible acheivements
  \item Local performance tracking 
  \item Library management
\end{itemize}

\subsection{Out of Scope}
What we should skip:

\begin{itemize}
  \item Online multiplayer services
  \item DRM bypassing
  \item Real time overlay injection
  \item Console libraries
\end{itemize}
