\section{Introduction}
Framing the problem space.

\subsection{Background}
\begin{itemize}
  \item Growth of digital game distribution
  \item Multiple launchers becoming unavoidable
  \item Lack of cross-platform analytics
\end{itemize}

\subsection{Problem Statement}
Existing game launchers operate in isolation, resulting in fragmented game libraries, inconsistent tracking, and limited insights into gameplay habits and system compatibility. Current recommendation systems rely primarily on purchase history and popularity metrics rather than actual user behavior and contextual factors like mood or avaialable time.

\subsection{Motivation}
\begin{itemize}
  \item Personal frustration
  \item Aggregation of all games regardless of platform
  \item Need for unified stats
  \item Learning opportunity (systems + UI + data)
\end{itemize}
